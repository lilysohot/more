## 🚀 快速开始

### 方式一：Docker 一键启动

```bash
# 1. 进入项目目录
cd more

# 2. 配置环境变量
cp backend/.env.example backend/.env

# 3. 一键启动
docker-compose up -d

# 4. 查看日志
docker-compose logs -f
```

**访问地址**：
- 🌐 前端：http://localhost:3000
- 🔧 后端 API: http://localhost:8000
- 📖 API 文档：http://localhost:8000/docs
