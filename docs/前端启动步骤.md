# 前端服务启动步骤

## 前提条件

确保已安装以下软件：

### 1. Node.js

- **版本要求**: 18+（推荐 20+）
- **下载地址**: https://nodejs.org/
- **验证命令**: `node --version`

### 2. pnpm（包管理工具）

**安装方法（任选其一）**：

**方法 A - 使用 npm 安装（推荐）**：
```bash
npm install -g pnpm
```

**方法 B - 使用 PowerShell 安装**：
```powershell
iwr https://get.pnpm.io/install.ps1 -useb | iex
```

**验证安装**：
```bash
pnpm --version
```

> 💡 **提示**：如果安装后提示"无法识别 pnpm 命令"，请重启终端或将其添加到 PATH 环境变量。

### 3. 替代方案

如果不想安装 pnpm，可以使用 **npm** 代替：

```bash
# 使用 npm 代替 pnpm
npm install      # 代替 pnpm install
npm run dev      # 代替 pnpm dev
```

## 启动步骤

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖

首次运行时需要安装依赖：

```bash
pnpm install
```

**说明**：
- 如果下载速度慢，可以配置国内镜像源
- 安装完成后会生成 `node_modules` 目录

### 3. 启动开发服务器

```bash
pnpm dev
```

**启动成功后会显示**：
```
  VITE v5.0.12  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

### 4. 访问前端页面

浏览器打开：http://localhost:5173

## 常见问题

### 1. 端口被占用

如果 5173 端口被占用，Vite 会自动使用下一个可用端口（5174、5175...）

### 2. 依赖安装失败

```bash
# 清除缓存后重新安装
pnpm clean
pnpm install
```

### 3. Node 版本不兼容

确保 Node.js 版本 >= 18：

```bash
node --version
```

如版本过低，请升级 Node.js。
